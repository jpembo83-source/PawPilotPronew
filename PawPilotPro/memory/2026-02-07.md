## What We Built Today

### Mission Control - A Desktop-Style UI for Orchestrating Jeff

A full-stack application for sending prompts to Jeff and monitoring runs:

**Frontend (React + Tailwind + Framer Motion)**
- Cyberpunk dark theme with cyan/purple accents
- Orbitron font for headers
- Glowing effects, glass-morphism
- Screens: Dashboard, Prompts, Runs, Console, Health, Incidents, Settings

**Backend (Node.js Bridge Server)**
- Serves static frontend + API on port 18800
- Proxies to OpenClaw Gateway via `/tools/invoke`
- Uses `sessions_spawn` to execute prompts in isolated sessions
- SSE streaming for real-time updates
- Token-based authentication

### Architecture
```
Mission Control UI (Browser)
       ↓
Bridge Server (:18800) - serves UI + API
       ↓  
OpenClaw Gateway (:18789) - /tools/invoke
       ↓
Jeff (sessions_spawn) - executes prompts
```
